# Find Skills

This skill allows an agent to search for and discover new skills that can be added via the `npx skills add` command.

## Instructions

1. Identify the need for a new skill.
2. Search available repositories (e.g., vercel-labs/agent-skills, supabase/agent-skills).
3. Analyze the SKILL.md of potential candidates.
4. Recommend the installation of the most relevant skill.
